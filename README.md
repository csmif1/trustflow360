# ILIT Management System

A comprehensive trust administration platform for managing Irrevocable Life Insurance Trusts (ILITs).

## Features

- Attorney Dashboard with multi-tab interface
- Premium payment tracking and receipts
- PDF receipt generation
- Email automation for payment confirmations
- Annual tax reporting with Form 1041 worksheets
- Bulk payment CSV upload
- Trustee portal for beneficiaries

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Supabase
- jsPDF for document generation

## Getting Started

```bash
npm install
npm run dev
```

## Database

This project uses Supabase for the backend. Configure your Supabase URL and anon key in the environment variables.
